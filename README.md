# Reliance Stock Price Forecasting

## Project Overview

This project focuses on forecasting the stock prices of Reliance Industries using Time Series Analysis and Machine Learning techniques.

The project includes:

* Data preprocessing
* Stationarity testing using ADF Test
* ACF & PACF analysis
* ARIMA forecasting
* Moving Average analysis
* Random Forest Regressor
* XGBoost Regressor
* Future stock price prediction


## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Statsmodels
* Scikit-learn
* XGBoost


## Models Implemented

### 1. ARIMA Model

Used for statistical time-series forecasting after making the data stationary.

### 2. Random Forest Regressor

Used lag features to capture historical stock price patterns.

### 3. XGBoost Regressor

Implemented boosting-based regression for improved predictive performance.


## Evaluation Metrics

The models were evaluated using:

* RMSE (Root Mean Squared Error)
* MAE (Mean Absolute Error)
